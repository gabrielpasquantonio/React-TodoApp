📋 React To-Do App

A simple, volatile to-do list application built with React JS and Hooks 📋





🛠️ Technologies
React
FireBase
⚙️ Requirements
Git
Node.js
NPM

🚀 Installation
$ git clone https://github.com/gabrielsanttana/react-todolist
$ cd react-todolist
$ npm install
$ npm start
The application will open in the browser on http://localhost:3000
